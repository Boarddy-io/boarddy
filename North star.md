# The Goal

Boarddy is not trying to become another productivity app.

Boarddy is trying to become:

"The Input Layer of Computing."

People spend their digital lives doing four things:

* Typing
* Copying
* Pasting
* Searching

Boarddy owns all four.

---

# The Core Insight

People think their problem is typing.

It isn't.

Their problem is memory.

Examples:

"I copied it yesterday."

"I typed it before."

"I saw it somewhere."

"I saved it but can't find it."

Boarddy solves memory through input.

# The Killer Feature

Not AI.

Not autocomplete.

Not voice typing.

Not sync.

The killer feature is:

Quick Paste

```text
Ctrl + Shift + V
```

The day a user starts using Quick Paste 20+ times daily, they become difficult to lose.

Everything should optimize toward this behavior.
