# Boarddy

> The Input & Memory Layer for Your Computer

Boarddy is a lightweight desktop productivity platform that upgrades how you type, copy, paste, search, remember, and interact with text across your entire computer.

Clipboard history was only the beginning.

Boarddy combines:

* Smart Clipboard Memory
* Quick Paste
* Autocomplete
* Autocorrect
* Personal Dictionary
* Notes
* Multilingual Typing
* Keyboard Gestures
* Predictive Key Selection (PKS)
* Developer Productivity Tools

into a single local-first experience designed to eliminate friction between thought and text.

---

## Core Features

### Clipboard Memory

Boarddy automatically remembers copied content.

* Search clipboard history
* Favorite entries
* Pin important items
* Metadata support
* Quick retrieval

---

### Quick Paste

Press:

Ctrl + Shift + V

Search your clipboard history instantly.

Find what you need.

Paste it immediately.

No mouse required.

---

### Predictive Key Selection (PKS)

Autocomplete without friction.

Accept suggestions using:

* Tab
* Right Arrow

Choose alternatives using:

* Numbers (1-9)

Example:

1. communication
2. community
3. commitment

Press:

2

Instantly inserts:

community

---

### Adaptive Learning

Boarddy learns your typing patterns.

Frequently selected suggestions automatically move higher in rankings.

Your keyboard becomes personalized over time.

---

### Personal Dictionary

Teach Boarddy your language.

Examples:

* Boarddy
* HunaPay
* EcoSynergy
* Cloakey
* HunaRemit

Custom words are never incorrectly autocorrected.

---

### Notes

Turn temporary information into permanent knowledge.

Convert clipboard items into searchable notes instantly.

---

### Multilingual Support

Use any language installed on your operating system.

Switch naturally between languages.

Type without interruptions.

---

### Keyboard Productivity Engine

Boarddy introduces desktop-first keyboard productivity features inspired by the best ideas from modern mobile keyboards and developer workflows.

Examples:

* Smart autocomplete selection
* Fast deletion actions
* Keyboard-driven navigation
* Quick clipboard access
* Gesture-based productivity

---

### Developer Mode (Future)

A productivity layer inspired by Vim, modern IDEs, and power-user workflows.

Planned features:

* Modal editing
* Text motions
* Chord shortcuts
* Universal editing commands
* Macro system
* Command palette

The goal is not to clone Vim.

The goal is to make advanced editing accessible without a steep learning curve.
