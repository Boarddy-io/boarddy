# Boarddy Master Document

## Overview

Boarddy is a desktop-first personal input and memory companion that helps users type faster, make fewer mistakes, manage clipboard history, store reusable content, and retrieve information effortlessly.

Unlike traditional keyboards, clipboard managers, or text expansion tools, Boarddy combines typing assistance, clipboard intelligence, note capture, language support, and personal memory into a unified experience.

Boarddy sits between the user and every application they use, becoming a trusted productivity layer across the operating system.
